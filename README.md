# Lifterico Delivery App

Lifterico is a smart delivery tracking and management system designed for the Nigerian market. It connects Small and Medium Enterprises (SMEs), logistics businesses, and dispatch riders to provide a transparent, secure, and efficient last-mile delivery solution.

## Key Features

- **Real-time GPS Tracking:** Monitor deliveries as they happen.
- **Role-based Dashboards:** Custom experiences for Admins, SMEs, Logistics companies, and Riders.
- **Proof of Delivery:** Secure confirmation using OTP and photo verification.
- **Fleet Management:** Logistics companies can manage and monitor their riders.
- **Analytics:** Business insights and revenue tracking for all stakeholders.

## Getting Started

### Prerequisites

- Node.js (Latest LTS)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env.local` file with your database and authentication credentials.
4. Run the development server:
   ```bash
   npm run dev
   ```
